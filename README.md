<h2>Overview</h2>
<p>capitalcarefull is a versatile developer toolkit that provides a comprehensive backend and frontend architecture for managing customer leads, employee data, and interactions across multiple platforms. It streamlines development with modular components, secure authentication, and an intuitive admin panel.</p>
<p><strong>Why capitalcarefull?</strong></p>
<p>This project aims to simplify building scalable, cross-platform applications for healthcare and financial services. The core features include:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0">🛠️ <strong>API-Driven Architecture:</strong> Robust RESTful APIs for seamless data management.</li>
<li class="my-0">🎨 <strong>Cross-Platform Support:</strong> Web, mobile, and desktop integrations with platform-specific configurations.</li>
<li class="my-0">⚙️ <strong>Modular Design:</strong> Scalable components for easy customization and extension.</li>
<li class="my-0">🔒 <strong>Secure Authentication:</strong> JWT-based middleware for protected routes.</li>
<li class="my-0">📊 <strong>Rich Admin Panel:</strong> React, Vite, TailwindCSS-based interface for efficient management.</li>
<li class="my-0">📁 <strong>Structured Data Models:</strong> Clear schemas for leads, employees, calls, and history.</li>
</ul>

<h2>Getting Started</h2>
<h3>Prerequisites</h3>
<p>This project requires the following dependencies:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0"><strong>Programming Language:</strong> Dart</li>
<li class="my-0"><strong>Package Manager:</strong> Npm, Pub, Cmake, Gradle</li>
</ul>
<h3>Installation</h3>
<p>Build capitalcarefull from the source and install dependencies:</p>
<ol>
<li class="my-0">
<p><strong>Clone the repository:</strong></p>
<pre><code class="language-sh">❯ git clone https://github.com/Himanshu2002Singh/capitalcarefull
</code></pre>
</li>
<li class="my-0">
<p><strong>Navigate to the project directory:</strong></p>
<pre><code class="language-sh">❯ cd capitalcarefull
</code></pre>
</li>
<li class="my-0">
<p><strong>Install the dependencies:</strong></p>
</li>
</ol>
<p><strong>Using <a href="https://www.npmjs.com/">npm</a>:</strong></p>
<pre><code class="language-sh">❯ npm install
</code></pre>
<p><strong>Using <a href="https://dart.dev/">pub</a>:</strong></p>
<pre><code class="language-sh">❯ pub get
</code></pre>
<p><strong>Using <a href="https://isocpp.org/">cmake</a>:</strong></p>
<pre><code class="language-sh">❯ cmake . &amp;&amp; make
</code></pre>
<p><strong>Using <a href="https://gradle.org/">gradle</a>:</strong></p>
<pre><code class="language-sh">❯ gradle build
</code></pre>
